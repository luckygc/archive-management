# Triage 标签

| 技能角色 | GitHub 标签 | 含义 |
| --- | --- | --- |
| `needs-triage` | `needs-triage` | 等待维护者评估 |
| `needs-info` | `needs-info` | 等待报告者补充信息 |
| `ready-for-agent` | `ready-for-agent` | 规格完整，可交给自动化 agent |
| `ready-for-human` | `ready-for-human` | 需要人工实现或决策 |
| `wontfix` | `wontfix` | 确认不处理 |

技能提及角色时，使用对应的 GitHub 标签。若仓库以后调整标签名称，只修改本表右侧标签列。
