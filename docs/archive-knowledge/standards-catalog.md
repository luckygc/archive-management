# 档案行业规范目录

本文记录首批建议纳入项目知识库的法规、国家标准、档案行业标准和专题规范。采纳结论分为三类：

- **优先理解**：和当前项目核心模型、接口、流程或验收高度相关。
- **按场景引用**：专题能力出现时再进入 OpenSpec。
- **暂不采纳**：和当前系统边界距离较远，先不写入项目合同。

## 暂不放入首批实现合同的领域

- 档案馆建筑、库房设备、温湿度、虫霉防治等实体保管规范。
- 艺术档案、电影胶片、地名档案、展览、缩微复制等垂直领域规范。
- 光盘、硬盘离线存储等物理介质规范，除非后续交付要求明确。

## 采纳规则

1. 先确认标准是否适用于当前业务场景。
2. 再把项目需要遵守的具体要求迁入 OpenSpec。
3. 只有当要求是协作、架构、工具链或代码风格时，才写入 `AGENTS.md`。
4. 代码实现必须引用 OpenSpec 或项目文档，不直接引用“某标准应该如此”的口头判断。
