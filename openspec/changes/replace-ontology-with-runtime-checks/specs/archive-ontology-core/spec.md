## REMOVED Requirements

### Requirement: 字段语义注册
**Reason**: 不同机构的真实字段已经由固定字段和分类元数据定义，额外字段语义注册形成第二套映射且没有进入业务事务。

**Migration**: 当前 `0.0.1` 不迁移旧字段语义数据；运行时约束和规则直接使用稳定固定字段代码、分类编码与字段编码重新定义。

### Requirement: 移除通用本体配置
**Reason**: 本 change 完成现行本体能力的整体删除，不再需要以“移除通用本体”作为长期业务能力存在。

**Migration**: 删除现行本体/字段语义 API、页面、权限文案、代码包、稳定规格、旧 DDL 和相关已归档 change，不保留历史兼容内容。
