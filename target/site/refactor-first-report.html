Refused Parent Bequest

Problem:Child class is large and often complex, but doesn't override or use many of the parent class's methods
Solution:Do one or more of the following:
- Extract the child class into a separate class. Move the methods that are used from the parent class into the child class.
- Make unused protected members private in the parent class.
- If a parent class has multiple children, move methods not used by all descendants to another class.

Shotgun Surgery
